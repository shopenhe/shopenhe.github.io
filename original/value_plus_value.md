# Value Plus Value

>**Tags**: Java<br>**Last Modify**: 2024/01/11 15:54

## Question

```
1 + '_' + 2 = ?
1 + "_" + 2 = ?
```

## Answer

```
1 + '_' + 2 = 98
1 + "_" + 2 = 1_2
```

## Thinking

It's a **interesting** question, we should be **careful**. If we want append String, also could use `StringBuilder` in Java.
